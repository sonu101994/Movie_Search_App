# 🎬 Movie Parlour

A simple and responsive **movie browsing web app** built using React.  
Users can explore movies, view details, watch trailers, and manage favorites.

---

## 🚀 What this project has

- Browse trending & popular movies  
- Movie detail page with full information  
- Watch trailers (YouTube integration)  
- Add / remove favorites  
- Favorites saved in localStorage  
- Smart image fallback (no poster → icon)  
- Responsive design (mobile friendly)  
- Clean and modern UI  

---

## 🛠️ Tech Used

- React  
- React Router  
- Bootstrap  
- React Toastify  
- React Icons  
- TMDB API
